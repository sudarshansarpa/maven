## ðŸ”§ How SSH Works:

1. You run a command like:

   ```bash
   ssh user@hostname
   ```

2. SSH:

   * Initiates a **handshake**.
   * **Authenticates** using password or key.
   * **Encrypts** the entire session.

3. Once connected, you can run shell commands as if you're sitting at the remote machine.
